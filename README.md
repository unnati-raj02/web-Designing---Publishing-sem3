# Web Designing & Publishing - Semester 3

This repository contains my HTML and CSS practice files for the **Web Designing & Publishing** subject in Semester 3.

## About the Subject
Web Designing & Publishing is a Semester 3 subject where we learn **basic web development**, including HTML structure, CSS styling, and responsive design techniques.

## Project Files

### 1. Basic HTML Page (`index.html`)
- Demonstrates **basic HTML structure**
- Includes **headers, paragraphs, images, and links**
- Designed to **practice HTML commands** and understand web page layout

### 2. Responsive Navigation Bar (`navbar.html` + `navbar.css`)
- Demonstrates a **practical use of HTML and CSS**
- Creates a **responsive navigation bar** 
- Designed to **practice CSS styling and layout principles**

## How to View
- Open any `.html` file in a web browser to see the web pages in action.
- CSS files are linked to the HTML pages, so they must stay in the same folder.
